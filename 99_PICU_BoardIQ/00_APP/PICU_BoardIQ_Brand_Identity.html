5 — Brand Guidelines (in brief)

Apply these rules everywhere

  • No emojis in the UI. Use only icons from this set. Emojis live in user-entered notes if at all — never in chrome.
  • Stroke width 1.75pt. Never mix stroke weights. Don't fill line icons.
  • Tinted square icon backgrounds. 38×38 rounded squares, background colour at 8–10% alpha of the icon's tint. This is what differentiates this look from generic Material/Bootstrap.
  • Navy is the brand colour. Blue is for primary actions only. Teal accents reserved for ICU/monitoring features.
  • Type: Inter (or system sans). Headings 700–800; body 400–500. Letter-spacing −0.02em on H1/H2.
  • Cards: 12px radius, 1px border at #E2E8F0, subtle shadow on hover only (0 6px 18px rgba(10,37,64,.06)).
  • Semantic colour: green = correct, amber = warning/exam timer, coral = error/incorrect. Never use blue or navy for these meanings.
